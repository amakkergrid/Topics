Converting a decimal to integer with rounding

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a program in Java which receives a decimal number x as input. Your task is to convert the decimal number into an integer and print the result. If the decimal part of the number is equal to or greater than 0.5, round the number up to the nearest whole number, otherwise round down.
Sample Input:
3.2

Sample Output:
3


Sample Input:
4.8

Sample Output:
5



Memory limit: 256 MB
Time limit: 5 seconds
