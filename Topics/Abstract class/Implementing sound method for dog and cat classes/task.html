Implementing sound method for dog and cat classes

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Consider a zoo management system where you are tasked to create an abstract class 'Animal' with a method 'sound()' that will return the sound made by that specific animal. You should define two subclasses: 'Dog' and 'Cat' that implement this method. The sound of a 'Dog' should be 'Bark!' and for 'Cat' it should be 'Meow!'. You get the name of the animal as an input string ('Dog' or 'Cat'). Write a program that creates an instance of the appropriate class based on the input and returns the sound made by the animal.
Sample Input:
Dog

Sample Output:
Bark!
