Calculating the area of a circle using class methods

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Create a Java class named 'Circle'. It should only have one member variable - 'radius' of type double. This class should have an instance method 'calculateArea' that returns the area of the circle. The area of a circle is calculated using the formula PI*radius*radius. You will be provided with a double value representing the radius of a circle. Your task is to create an object of class 'Circle', set its radius, use the 'calculateArea' method to calculate and print the area of the circle.
